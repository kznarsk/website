Zobrollo is a 2d minimalistic top-view racing game programmed in C using the Allegro library. At this point of development you can only play in time trial. The game also has a record keeping system and you can save a race you have played and race against your older selve.

Installation

On a Unix based system you will first need to compile the source code. You can get the source code by downloading the zip file or cloning it from my github. You should also make sure you have Allegro installed. Then go to the directory of the source and run make. If everything went well an executable will appear that you can run. At the moment I haven't tried compiling for windows, so it will probably not work. This I intend to change in the future.

Running the game

To play the game you have to specify the track you want to play on. At the moment there are only two tracks Long Straight and example. You can find these files in the directory of the source. On a Unix based system you can run the game by going to the directory of executable and running the comand: ./race [OPTIONS] [Track file]. You should replace [Track file] by the path of the track. Then there are some optional flags you can use:
  • -r [record file]: Reads and writes records to record file
  • -s [ghost file]: Store the race you are currently playing in ghost file.
  • -p [ghost file]: Race against the ghost stored in ghost file.
