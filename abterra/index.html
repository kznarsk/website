The Abterra I

Go back to main page
The Abterra I is a computer built in minecraft without the use of any command blocks or mods. This computer differs from other minecraft computers in a few ways:

  • Small and simple instruction set.
  • Instruction and data stored in the same place.
  • Programs can be written from the control room.
To actually figure out how the machine works and how the computer works and how to program it download the manual.

If you want to improve the project go to my Github.
