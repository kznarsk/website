Zobrollo

Go back to main page
Zobrollo is a 2d minimalistic top-view racing game programmed in C using the Allegro library. At this point of development you can only play in time trial. It also keeps of track of records you have made.

How to install

GNU/Linux

The installation process is different per distribution.

Arch Linux

  1. Install the package `zobrollo` from the AUR.
  2. Run the comand `zobrollo` in your terminal and the game should start

Other Distributions

You can't install Zobrollo for the other distributions at the moment. But if you are running GNU/Linux it is quite easy to build it yourselve.

Debian/Ubuntu

  1. Download the .deb file
  2. Install it as follows:
    3. Go to the directory you downloaded it to and run the command:
    sudo dpkg -i zobrollo-0.2.0_amd64.deb
  3. You must also install the dependencies. You do this by running this command directly after the previous command.
    sudo apt-get -f install
  4. Run the comand `zobrollo` in your terminal and the game should start
